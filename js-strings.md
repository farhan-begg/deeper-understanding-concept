String Methods
- charAt()
    * Returns the character at the specified index

- charCodeAt()
    * Returns a number indicationg the Unicode value of the character at the given index.

- lastindexOf()

- localeCompare()

- match()
    * Used to match a regular expression against a string

- replace()

- search()

- slice()

- split()

- substr()

- substring()
